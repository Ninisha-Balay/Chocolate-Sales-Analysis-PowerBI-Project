# Chocolate-Sales-Analysis-PowerBI-Project
**📊 Chocolate Sales Analysis Dashboard (Power BI)**

**📘 Overview**

This project features an interactive Power BI dashboard designed to analyze chocolate sales performance across key business metrics. The dashboard highlights trends in revenue, profit, product performance, shipments, and geographic sales contribution.
The goal of this project is to showcase end-to-end Business Intelligence (BI) skills, including data preparation, data modeling, and visualization.

**🎯 Project Objectives**

- Analyze total sales, profit, boxes sold, and shipments

- Compare Current Year (CY) vs Previous Year (PY) performance

- Identify top-performing products and salespersons

- Understand country-wise sales contribution

- Explore shipment patterns and frequency

- Build an interactive, user-friendly dashboard

**📁 Dataset Information**

The dataset used is titled “Sample Chocolate Shipments Data – Apr 2025”, sourced from user-shared data on Scribd. It includes shipment-level sales information such as product, salesperson, geography, date, boxes, amount, profit, and order status.

**🛠️ Tools & Technologies**

- Power BI Desktop

- Power Query

- Star Schema Data Modeling

- Excel / CSV source files

**📐 Data Modeling**

- A Star Schema approach was used with:

- Fact Table: Sales/Shipments

- Dimension Tables: Product, Salesperson, Country, Date

This structure supports efficient relationships and clear, organized analysis.

**📊 Dashboard Features**

- KPI cards (Sales, Profit, Boxes, Shipments)

- Monthly CY vs PY comparison charts

- Top 5 Products and Top 5 Salespersons

- Shipment distribution visualization

- Geographic sales breakdown

- Product profitability insights
